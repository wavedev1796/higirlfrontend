# 🌸 Hi Girl

> **"Conecta con mujeres que realmente comparten tu estilo de vida e intereses."**
> A safe space to be you.

Hi Girl es una plataforma diseñada exclusivamente para conectar mujeres según afinidades personales, estilo de vida, intereses y ubicación geográfica. Este repositorio contiene la interfaz web (Frontend) de la primera fase del proyecto (MVP Web).

---

## 🚀 Stack Tecnológico

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS

---
