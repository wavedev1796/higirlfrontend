# HG-F3-001 - Pantalla de Login Web

## Resumen

Se reemplazo la pantalla inicial generada por `create-next-app` por una pantalla de login propia de Hi Girl, alineada al branding definido en la documentacion tecnica del proyecto.

## Objetivo

Crear una primera experiencia web moderna para que la usuaria pueda acceder a Hi Girl mediante correo electronico y contrasena.

## Alcance Implementado

- Pantalla principal `/` convertida en login.
- Diseno responsive para desktop y mobile.
- Uso de paleta oficial del branding:
  - `#E7C6FF`
  - `#D6BCFA`
  - `#FFC8DD`
  - `#FFF9FB`
  - `#6B7280`
  - `#1F2937`
- Logo visual construido con CSS.
- Panel de marca con mensaje de bienvenida.
- Formulario con campos de correo y contrasena.
- Estados visuales de carga, exito y error.
- Link hacia la pagina de registro `/register`.

## Archivos Modificados

- `app/page.tsx`
- `app/layout.tsx`
- `app/globals.css`

## Criterios de Aceptacion

- La ruta `/` muestra la pantalla de login de Hi Girl.
- El diseno ya no muestra contenido por defecto de Next.js.
- La pantalla usa la paleta oficial del proyecto.
- El formulario se adapta a pantallas pequenas.
- El link "Crear cuenta" navega hacia `/register`.
